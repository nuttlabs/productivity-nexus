3 AI Productivity Hacks That'll Save You an Eternity

Here are three AI-adjacent productivity hacks I'm loving and sure you will, too:

1. Don't type. Speak!

Speaking is 4x faster than typing.

  • Wispr transcribes and polishes my spoken word across apps—desktop and mobile.
  • With Voicenotes, I can ramble endlessly about any idea, then it'll tidy and format it for any need.

2. Save prompts as snippets.

  • I use Raycast to make my reused prompts available in a few taps of the keyboard. They're configured with variables for the inputs that change.
  • Text Blaze is an interesting new browser extension that serves the same function.

3. Chat with multiple bots.

No AI chatbot is best for every prompt. If you want the most useful output, prompting multiple chatbots is a must.

  • Tools like ChatHub and Magai make it easy.
